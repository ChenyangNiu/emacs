Emacs for Windos
1. GNU Emacs for Windows can be downloaded from the main GNU FTP server, http://ftp.gnu.org/gnu/emacs/windows/
2. Unzip the zip file preserving the directory structure
3. Put the profiles "Emacs_for_Win\*" to C:/Users/Yourname/AppData/Roaming
4. Create a desktop shortcut to bin\runemacs.exe, and start Emacs by double-clicking on that shortcut's icon.
